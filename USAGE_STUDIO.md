# Studio Kasperphi — guide pour Phil

**En une phrase :** le Studio est votre tableau de bord privé pour publier, modifier, ranger et supprimer toutes vos créations et tous vos médias (images, chansons, vidéos) — sans toucher à une ligne de code.

## Se connecter
1. Allez sur **votre-site/studio** (ex. `https://kasperphi.naskaus.com/studio`).
2. Tapez votre **code** à 4 chiffres.
3. Vous arrivez sur le tableau de bord.

> Bouton **Déconnexion** en haut à droite quand vous avez fini.
> Bouton **Voir le site ↗** pour ouvrir le site public dans un nouvel onglet.

## Les 3 onglets

### 1. Créations
La liste de toutes vos œuvres. Pour chacune :
- **Interrupteur vert « Publié / Brouillon »** → c'est le cœur : **Publié = visible sur le site**, **Brouillon = rangé, invisible**. Un clic suffit.
- **↑ ↓** → changer l'ordre d'affichage.
- **Modifier** → ouvre la fiche.
- **Suppr.** → supprime (avec une demande de confirmation).

**Ajouter une œuvre :** bouton **＋ Nouvelle création**. Remplissez :
- *Titre*, *Rubrique* (One Mon Show, Chansons, Images, Textes, Pochettes),
- *Résumé* (la petite phrase sur la carte),
- *Texte* (laissez une ligne vide entre les paragraphes),
- *Image de couverture* et *Audio / vidéo* (à choisir dans votre bibliothèque),
- l'interrupteur **Diffusion** (Publié / Brouillon),
- **Enregistrer**.

### 2. Bibliothèque média
Tous vos fichiers. Pour ajouter : **glissez-déposez** vos fichiers dans le cadre, ou cliquez pour les choisir. Accepte **images, audio (chansons, One Mon Show), vidéos**.
Pour chaque média : **Renommer**, **Lien** (copie l'adresse), **Suppr.**.

> Pour mettre une chanson ou une vidéo dans une création : uploadez-la d'abord ici, puis choisissez-la dans la fiche de la création (champ *Audio / vidéo*).

### 3. Réglages
Les textes du site que vous pouvez changer vous-même : titre d'accueil, sous-titre, phrase d'accroche, bloc « Avant d'entrer », texte du Manifeste, email de contact. Modifiez puis **Enregistrer les réglages**.

## Bon à savoir
- **Tout est instantané** : dès que vous publiez ou modifiez, le site public est à jour (rafraîchissez la page).
- **Brouillon = sécurité** : préparez une œuvre tranquillement en brouillon, publiez quand elle est prête.
- **Rien n'est perdu** : un média supprimé disparaît du stockage ; une création supprimée demande confirmation.
- **Téléphone** : le Studio marche aussi très bien sur mobile.

## En cas de souci
- « Je publie mais je ne vois rien » → rafraîchissez la page publique (Ctrl/Cmd + R).
- « Mon image n'apparaît pas dans la création » → vérifiez qu'elle est bien dans la Bibliothèque média, puis re-sélectionnez-la dans la fiche.
- Code oublié / à changer → demandez à Seb.
